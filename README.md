# SmartCloudBot v5.9

**بوت تداول ذكي للذهب (GC=F) على إطار 5 دقائق**  
مدعوم بـ **Layered Logic** + **Nenuo AI** + Stop Loss و Take Profit

---

### المميزات
- استراتيجية Scalping متقدمة باستخدام Layered Logic (3 طبقات لتقليل الإشارات الكاذبة)
- حساب Stop Loss و Take Profit تلقائي بناءً على ATR
- Nenuo AI للتصحيح الذاتي والتحسين المستمر
- إشعارات فورية على تليجرام
- نظام تسجيل الصفقات وحساب الربح/الخسارة
- مصمم خصيصاً للعمل على Railway

### الملفات الأساسية
- `main.py` → الكود الرئيسي للبوت
- `requirements.txt` → المكتبات المطلوبة
- `nixpacks.toml` → إعدادات التشغيل على Railway

### كيفية الاستخدام
1. ارفع الملفات على GitHub
2. اربط الـ Repository بـ Railway
3. أضف المتغيرات في Railway:
   - `TELEGRAM_TOKEN`
   - `CHAT_ID`
   - `GROQ_API_KEY`
